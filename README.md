qiwi-gate
=========

Gate emulates the qiwi system (REST Protocol).

Requirements

php >=5.3.0
Laravel Framework 4.1.*
MySQL Database
Laravel queue driver configuration
User auth identifier in your web project

Uses

bootstrap cdn
jquery cdn

Installation

Composer

{
    "require": {
        "fintech-fab/qiwi-gate": "dev-master"
    },
}
